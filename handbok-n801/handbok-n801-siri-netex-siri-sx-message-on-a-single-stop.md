## A message is shown whenever a certain stop is used <a id="SIRI-SX-Messageonasinglestop-Amessageisshownwheneveracertainstopisused"></a>

The message is shown on all vehicles serving the stop in question, and shown in searches to-, from-, via- or passing the stop, including those operated by different agencies/authorities.

If the message is relevant to a specific Quay, it should be referenced directly.

If the specific Quay is unknown, or the message is relevant for all Quays at the stop, the reference is to the StopPlace.
